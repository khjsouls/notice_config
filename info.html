위드펀 사용 매뉴얼


사전 준비 사항

  1. 골드 엔젤 준비
  2. 주얼팩 준비
  3. PowerToys 프로그램 설치
    - 위드 창을 정위치 시키기 위한 프로그램입니다. (microsoft 지원)
    - PowerToys 다운로드
  4. WYD2FC Fun 활성 코드 발급 & 다운로드
    - 발급 요청 문의 : 카카오톡 오픈채팅 위드펀
    - WYD2FC Fun 다운로드

필수 설정 사항

  1. 위드 게임 해상도
    - 반드시 '800 x 600'으로 설정해주세요.
    - 다른 해상도로 설정하게되면 비정상적인 동작을 합니다.
    - 다른 해상도 지원은 추후 지원 예정입니다.
  2. 장비 창
    - 처음 위치를 유지해주세요.
  3. 창고 창
    - 좌상단 끝으로 이동시켜주세요.
    - 골드 엔젤은 창고 창 오른쪽 가장 아래칸에 위치시켜주세요.
  4. 영역 분할
    - PowerToys > FancyZones > 레이아웃 편집기를 이용해서 영역을 분할 설정해주세요.


    - 새 레이아웃 만들기를 해주세요. - 이미 생성한 레이아웃이 있다면 사용자 지정 레이아웃 수정으로 변경할수 있습니다.

    - 영역 주위 공간 비활성화로 변경해주세요.
    - 강조 표시 간격 '0px'로 설정


    - 영역 분할은 반드시 '800 x 626' 단위로 해주세요. (최대 6개 영역)
    - 위드 게임창은 최대 6개까지 지원됩니다.


    - 창 이동 방법은 영역 분할 설정을 모두 마친 후 위드 창을 Shift키를 누른 상태에서 해당 영역으로 드래그하여 이동시켜주세요.

WYD2FC Fun 설정법

WYD2FC-Fun 프로그램은 반드시 관리자 권한으로 실행해주세요.
- Shift + 마우스 우클릭하면 '관리자 권한으로 실행' 메뉴가 보입니다.

계정 설정
  • 가장 먼저 발급 받은 활성 코드를 입력하고 인증 받으세요.
  • 가방 정리할 캐릭터 계정 수를 선택 후 적용시켜주세요.
  • 아이디, 비밀번호, 2차 비밀번호, 캐릭터명, 계정 타입을 정확하기 입력해주세요.
    - 자동 로그인에 필요한 필수값입니다.
    - 아이디, 비밀번호, 2차 비밀번호는 '모든 설정값 저장'을 통해 암호화하여 저장됩니다.
  • 텔레그램 token, chat.id 입력란은 텔레그램으로 알림을 받기 위한 기능입니다.
    - 현재 준비중인 기능입니다.
    - PC 이름은 텔레그램 알림을 받을때 PC 이름을 구분하기 위한 값입니다.

매크로 설정

  1. - 초기 설정값으로 유지시 매시간 3분 간격으로 가방정리를 하게 됩니다.
    - ex) 9시 00분, 9시 03분, 9시 06분, ...

  2. - 보관할 아이템에 추가할 아이템을 선택하는 영역입니다.

  3. - 해당 캐릭터가 로그인할 채널을 선택합니다.
    - 로그인 버튼을 클릭하게되면 해당 캐릭터만 로그인을 시도합니다.
    - 현재는 게임매니아, 온게이트 계정은 로그인 지원이 불가합니다.

  4. - 계정 정보를 모두 입력하고 활성화된 캐릭터를 모두 로그인하게 됩니다.

  5. - 체크하게되면 해당 캐릭터는 '모든 계정 로그인' 기능에서 제외됩니다.

  6. - 레저렉트를 배운 상태라면 캐릭터가 사망시 부활을 시도합니다.

  7. - 체크하게되면 해당 캐릭터는 가방정리 대상 캐릭터에서 제외됩니다.

  8. - 상단의 '아이템 리스트'에서 선택한 아이템을 추가합니다.
    - 좌측 리스트에서 삭제할 아이템을 선택한 후 '-' 버튼으로 삭제합니다.

  9. - 가방의 아이템을 임의로 위치 변경을 할 경우 사용해주세요.
    - 해당 캐릭터의 가방을 처음부터 다시 정리합니다.

  10. 해당 캐릭터만 즉시 가방정리를 합니다.

  11. - 매크로 상태를 나타냅니다.
    - 만료일은 '활성 코드'의 유효기간을 뜻하며 해당 기간까지 프로그램을 사용할수 있습니다.
