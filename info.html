위드펀 사용 매뉴얼


이럴 때 사용하세요!

  1. 잡템을 가방에 가득채워 팔아서 돈을 모으고 싶을 때
    - 골드엔젤 구입 비용을 엄청나게 절약 할수 있습니다.
    - 쓰레기 아이템은 바로 삭제 할수 있습니다.
  2. 필요한 아이템을 수집하고 싶을 때
    - 겹쳐지는 아이템은 알아서 겹치면서 수집합니다.
    - '가위바위보 쿠폰'은 자동으로 '승리 쿠폰'으로 만듭니다.
  3. 매주 로그인이 번거로울 때
    - 온게이트, 게임매니아 계정은 추후에 제공 예정입니다.
  4. 캐릭터가 안죽고 C.C가 잘 돌아가고 있는지 궁금할 때
    - 텔레그램 앱으로 캐릭터가 죽었을 때 알림으로 받을수 있습니다.

사전 준비 사항

  1. 골드 엔젤 준비
    - 가방을 자동으로 비울때 필요합니다.
  2. 주얼팩 준비
    - 사냥터에서 창고를 열어둬야해서 필요합니다.
  3. WYD2FC Fun 활성 코드 발급 & 다운로드
    - 발급 요청 문의 : 카카오톡 오픈채팅 위드펀
    - WYD2FC Fun 다운로드

필수 설정 사항

  1. 위드 게임 해상도
    - 반드시 '800 X 600'으로 설정해야 합니다.
    - 다른 해상도로 설정하게되면 정상적으로 동작하지 않습니다.

    - 반드시 창모드로 실행해주세요. UI > Mode > Windowed
  2. 장비 창
    - 반드시 처음 위치를 유지해야 합니다.
  3. 창고 창
    - 좌측 상단 끝으로 이동시켜둬야 합니다.
    - 골드 엔젤은 창고 창 오른쪽 가장 아래칸에 위치시켜둬야 합니다.
  4. 게임 창 위치 조정
    - 계정 수에 맞추어 창을 정위치로 이동시켜둬야 합니다.
    - '창 위치 / 타이틀 변경' 탭에 '이동' 기능을 이용해 주세요.

WYD2FC Fun 설정법

  • 관리자 권한으로 실행

    - 이미지 저장 및 설정 내용을 저장하기 위해 관리자 권한으로 실행합니다.
    - '게시자를 알 수 없는 이 앱이 디바이스를 변경할 수 있도록 허용하시겠어요?' 창에서 '예'를 눌러주세요.
    - 프로그램 최초 실행시에는 1~2분 가량 시간이 소요될수 있으니 잠시 기다려주세요.
  • 게임 창 최대 6개 지원
    - 5~6개 지원 가능한 윈도우 해상도: 2,400 X 1,878 이상
    - 3~4개 지원 가능한 윈도우 해상도: 1,600 X 1,252 이상
    - 2개 지원 가능한 윈도우 해상도: 1,600 X 626 이상
    - 1개 지원 가능한 윈도우 해상도: 800 X 626 이상

계정 설정
  1. 가장 먼저 발급 받은 '활성 코드'를 입력하고 '인증' 받은 후 사용해 주세요.
  2. 가방을 정리할 캐릭터 수를 선택하고 '적용'을 눌러주세요.
  3. 아이디, 비밀번호, 잠금 비밀번호, 캐릭터명, 계정 타입을 정확하게 입력해 주세요.
    - 자동 로그인에 필요한 필수값입니다.
    - 아이디, 비밀번호, 잠금 비밀번호는 암호화되며 서버에는 저장하지 않습니다.
  4. 텔레그램 token, chat.id 입력란은 텔레그램으로 알림을 받기 위한 기능입니다.
    - 'PC 이름'은 텔레그램 알림을 받았을때 쉽게 구분하기 위한 값입니다.

매크로 설정
  1. 시작
    - 기본값으로 '시작'하게되면 매시간 3분 간격으로 가방을 정리합니다.
    - 정상적으로 시작이 되면 버튼이 문구가 '작동중'으로 변경됩니다.
    - ex) 9시 00분, 9시 03분, 9시 06분, ...
  2. 정지
    - '작동중' 상태일때 언제든지 '정지' 버튼을 통해 매크로를 멈출수 있습니다.
    - 단, 이미 실행중인 스케쥴은 완료될때까지 정지되지 않습니다.
  3. 로그인
    - 먼저 '계정 설정'탭에 '아이디', '비밀번호', '잠금 비밀번호'를 입력해주세요. 해당 정보가 누락되어있다면 로그인 하지 않습니다.
    - 좌측에 선택한 서버로 로그인을 시도합니다.
    - 온게이트, 게임매니아 계정은 추후 지원 예정입니다.
  4. 모든 계정 로그인
    - 계정 정보를 모두 입력하고 활성화된 캐릭터를 모두 로그인하게 됩니다.
    - 온게이트, 게임매니아 계정은 추후 지원 예정입니다.
  5. 자동 로그인 안함
    - 체크해두면 해당 계정은 '모든 계정 로그인' 대상에서 제외됩니다.
  6. 부활 시도
    - '레저렉트' 스킬을 배운 상태라면 캐릭터 사망시 부활을 시도합니다.
  7. 가방 정리 안함
    - 체크해두면 해당 계정은 가방정리 대상에서 제외됩니다.
  8. 보관할 아이템
    - 상단의 '전체 아이템 목록'에서 선택한 아이템을 '+' 버튼으로 추가합니다.
    - 아이템 목록이 많은 경우 가방 정리하는데 시간이 많이 소요될수 있으므로 사냥터에 맞는 아이템들만 설정하는것을 권장합니다.
  9. 삭제할 아이템
    - 리스트 중에서 삭제할 아이템을 선택한 후 '-' 버튼으로 삭제합니다.
    - 아이템 목록이 많은 경우 가방 정리하는데 시간이 많이 소요될수 있으므로 사냥터에 맞는 아이템들만 설정하는것을 권장합니다.
  10. 가방 초기화
    - 가방에 있는 아이템을 임의로 변경한 경우에 초기화를 해줘야 정상 동작합니다.
  11. 가방 수동 정리
    - 해당 캐릭터만 즉시 가방을 정리합니다.
  12. 하단 상태 메세지
    - 매크로 현재 상태를 나타냅니다.
    - 만료일은 '활성 코드'의 유효기간을 뜻하며 해당 기간까지 프로그램을 사용할수 있습니다.
  13. 모든 설정값 저장
    - 모든 설정을 마친 후에 꼭 저장을 해주세요.

창 이동 / 타이틀 변경
  1. 이동
    - '실행중인 게임 리스트' 에서 선택한 게임의 타이틀을 변경하고 위치를 변경합니다.
  2. 변경
    - '실행중인 게임 리스트' 에서 선택한 게임의 타이틀을 '변경할 타이틀'에 입력한 값으로 변경합니다.
  3. 새로고침
    - '실행중인 게임 리스트'를 갱신합니다.
